# Support

## Getting help

- Use GitHub issues for bug reports and feature requests.
- Use pull requests for concrete fixes.
- Use the maintainers' private contact path for security-sensitive reports.

## Before opening an issue

Include:

- LuxReact version or commit
- React version
- adapter type or comm-layer details
- reproduction steps
- expected behavior
- actual behavior

## Scope

The maintainers can best support the public library surface in `src/` and documented demo setup in `README.md`.
