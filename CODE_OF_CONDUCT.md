# Code of Conduct

## Our standard

This project expects respectful, constructive collaboration.

Examples of expected behavior:

- Be respectful in technical disagreement.
- Focus reviews on code and behavior, not people.
- Share context, assumptions, and reproducible evidence.
- Accept feedback and correct mistakes quickly.

Examples of unacceptable behavior:

- Harassment, intimidation, or personal attacks.
- Discriminatory or demeaning language.
- Publishing private information without permission.
- Disruptive behavior that prevents productive collaboration.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that violate this Code of Conduct.

## Reporting

Report conduct issues privately to the maintainers through the security/support contact path documented in `SUPPORT.md`.
